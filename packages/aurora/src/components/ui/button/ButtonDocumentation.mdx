## Overview

Buttons are clickable elements that are used to trigger actions. They communicate calls to action to the user and allow users to interact with pages in a
variety of ways. Button labels express what action will occur when the user interacts with it.

### Button asChild

The `asChild` prop allows you to render the button as a child of another component.
The following example shows how to render a button as a child of an anchor tag.

```tsx title="Button asChild"
<Button asChild>
  <a href="/">Button</a>
</Button>
```

### Button Loading

The `loading` prop allows you to show a loading state on the button.
The following example using Tanstack Query shows how to render a button in a loading state.

```tsx title="Button loading"
const { isPending, isSuccess, isError, data, error } = useQuery({
  queryKey: ['posts'],
  queryFn: () => fetchPosts(),
})

<Button loading={isPending}>
  {isPending ? 'Loading...' : 'Button'}
</Button>
```

### Button Inverse

The `inverse` prop allows you to render the button with an inverse color scheme on a dark background.

```tsx title="Button inverse"
<Button inverse>
  Button
</Button>
```

The `leftIcon` prop allows you to render an icon on the left side of the button.

  leftIcon={<Icon name="Atom" />}
